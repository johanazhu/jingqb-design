# List 列表大组件

此组件用于各类（混合列表）展示的场景，如：多种类型的交互在同一个列表内，即有单选，又有复选，或者点击跳转等情况；

包括但不限于（单选，复选，点击跳转，银行卡，纯文字，输入框，图标）等；

## 群体操作——适用于大量数据

<Playground>
  {() => {
    const Temp = () => {
      return (
        <List
          bordered
          dataSource={data.data}
          renderItem={(item) => (
            <List.Item 
              renderIcon={item.icon && <Icon kind={item.icon} size="small" />}
              hasLink={item.link} 
              onClick={()=>{
                alert(item.id)
              }}>
              {item.text}
            </List.Item>
          )} /> 
      );
    };
    return <Temp />;
  }}
</Playground>

## List API
| 参数 | 说明 | 类型 | 可选值 | 默认值 |
| :--- | :--- | :---- | :--- | :---- |
| className | 列表组件样式名 | String  | - |  -  |
| children | 渲染的内容 | any  | - |  -  |
| style | 列表组件的样式 | object  | - |  -  |
| bordered | 是否展示上下变宽，默认展示 | `true` `false` |  `true`  |
| header | 列表组件头部自定义 | ReactNode  | - |  -  |
| footer | 列表组件尾部自定义 | ReactNode  | - |  -  |
| isLoading | 拉取数据时显示 loading 图标，数据拉取完毕显示内容 | boolean  | `true` `false` |  `false`  |
| dataSource | 数据源 | any  | - |  -  |
| renderItem | 当有数据源时，自定义渲染列表项 | ReactNode  | - |  -  |


## ListItem API
| 参数 | 说明 | 类型 | 可选值 | 默认值 |
| :--- | :--- | :---- | :--- | :---- |
| className | 列表项class | String  | - |  -  |
| children | 渲染的内容 | any  | - |  -  |
| style | 列表项的样式 | object  | - |  -  |
| hasLink | 是否有链接 | boolean  | `true` `false` |  `false`  |
| renderIcon | 渲染前图标 | ReactNode  | - |  -  |


## ListItem 事件

| 事件名称 | 说明 | 参数 |
| :--- | :--- | :--- |
| onClick | 点击列表项后的回调，前提是 hasLink 为 `true` | - |
