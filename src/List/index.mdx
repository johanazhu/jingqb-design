# List 列表大组件

此组件用于各类（混合列表）展示的场景，如：多种类型的交互在同一个列表内，即有单选，又有复选，或者点击跳转等情况；

包括但不限于（单选，复选，点击跳转，银行卡，纯文字，输入框，图标）等；

## 群体操作——适用于大量数据

<Playground>
  {() => {
    const Temp = () => {
      return (
        <List
          bordered
          dataSource={data.data}
          renderItem={(item) => (
            <List.Item 
              renderIcon={item.icon && <Icon kind={item.icon} size="small" />}
              hasLink={item.link} 
              onClick={()=>{
                alert(item.id)
              }}>
              {item.text}
            </List.Item>
          )} /> 
      );
    };
    return <Temp />;
  }}
</Playground>
